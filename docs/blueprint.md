# **App Name**: FestConnect

## Core Features:

- Event Discovery & Registration: Browse events, view details, and register seamlessly. Integrated with event scheduling and team logic.
- Digital Wallet: Top-up wallet via UPI, manage payments, and process refunds. Utilizes double-entry ledger for auditability.
- Merchandise Marketplace: Browse and purchase event-related merchandise. Includes cart, order management, and QR code pickup workflow.
- AI-Powered Chatbot Support: Provide real-time support for users with an AI-powered chatbot, assisting with event information, merchandise inquiries, and FAQs. The chatbot can use a tool for escalating complex issues to human support.
- Sponsor Branding & Advertising: Showcase sponsors through dedicated branding sections, logo displays, and campaign integration. Track metrics for sponsor engagement.
- Content Archive: Gallery of past fest photos/videos and CDN content from prior years with the possibility to filter by year or topic.
- Analytics Dashboard: Display analytics, metrics, live dashboards, reports.

## Style Guidelines:

- Primary color: Deep Purple-Brown (#312A41) to evoke sophistication.
- Background color: Dark Plum (#1A1625) to set a luxurious tone.
- Accent color: Metallic Gold (#D4AF37) to highlight key interactive elements and CTAs.
- Body text: 'Roboto', sans-serif, regular weight for readability.
- Headline font: 'Oswald', sans-serif, bold weight to grab attention.
- Use a set of minimalist vector icons with a thin stroke weight in Metallic Gold (#D4AF37).
- Employ a card-based layout with generous padding to give a sense of spaciousness and allow focus on content.
- Use subtle fade-in animations and smooth transitions to enhance user experience without being distracting.
